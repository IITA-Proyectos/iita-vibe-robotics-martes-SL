# 🗂️ Tests del Profe Franco

Carpeta de trabajo del profe. Organizada en subcarpetas:

## 📁 robot-codigo/
Código Pybricks listo para cargar al robot:
- `robotL.py` + `robotL.md` — movimientos en L con corrección de gyro
- `lecturaSensor.py` + `lecturaSensor.md` — lectura de sensores de color
- `seguidorLinea.py` + `seguidorLinea.md` — seguidor básico P
- `a_main.py` — seguidor de línea completo con zona de rescate adaptativa (Pybricks)
- `m_main.py` — seguidor de línea con HuskyLens y zona de evacuación (Spike Legacy)
- `nosubir.py` — código de trabajo, no pushear

## 📁 referencias/
Análisis técnicos profundos de sistemas de competición:
- `a_reference.md` — análisis del código Pybricks + zona de rescate adaptativa
- `m_reference.md` — análisis del código Spike Legacy + HuskyLens
- `comparacion_modelos.md` — comparativa entre enfoques

## 📁 clases/
Material de enseñanza y planificación:
- `plan_clase_hoy.md` — planificación de la clase
- `ejercicios_alumnos.md` — ejercicios para los chicos
- `ejercicios_clase_hoy.md` — ejercicios del día
- `guia_antigravity_alumnos.md` — guía de uso de la IA

## 📁 tests-ia/
Pruebas de código generado por distintos modelos de IA:
- `test-minimax.py`, `test-qwen.py`, `test-bigpickle.py`, etc.
- Documentación de resultados de cada modelo
