# Attachment patterns

## Core cycle questions

- where is mechanical home?
- can the attachment be zeroed at the table quickly?
- what angle or position means fully delivered?
- what partial reset is dangerous?

## Common failure modes

- attachment not fully re-homed before the next launch
- beam flex causing under-travel under load
- robot moving before the mechanism has settled
- double action due to repeated trigger calls
