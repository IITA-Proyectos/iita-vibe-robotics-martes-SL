# 🧠 Skills de Claude para Robótica

Los "Skills" son archivos de conocimiento especializado. Cuando le pedís código a una IA, podés pegarle estos archivos como contexto para que escriba mejor código.

## Skills disponibles

| Skill | Archivo | Qué cubre |
|-------|---------|-----------|
| 🧭 Giroscopio | `guia-giroscopio.md` | Inicialización, calibración, heading_correction, PID giros, anti-drift, troubleshooting |
| 🏆 Competencia | (en `docs/mejores-practicas-competencia.md`) | Diseño mecánico, arquitectura código, FLL/WRO/RoboCup |

## Cómo usar

1. Abrí el archivo `.md` que necesitás
2. Copiá el contenido
3. Pegalo como primer mensaje en Claude/ChatGPT antes de pedir código
4. La IA va a aplicar todo ese conocimiento automáticamente
